## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code, Sublime Text, etc.)
- For PHP projects: PHP server (XAMPP, WAMP, or built-in PHP server)
- For Python projects: Python 3.x installed

### How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sg-2003/SG.git
   cd SG
   ```

2. **Navigate to any project folder**
   ```bash
   cd Project_Name
   ```

3. **Open the project**
   - For HTML/CSS/JS projects: Open `index.html` in your browser
   - For PHP projects: Use a local server (XAMPP, WAMP, or `php -S localhost:8000`)
   - For Python projects: Run `python script_name.py`
