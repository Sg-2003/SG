CASCADE: It deals with precedence of style rules or CSS properties(i.e. the order in which properties are applied to an html element). It soilves conflict solutions.

When we define set of style rules for a specific tag in external, internal as well as inline styles the rules get cascade.

When we define set of style rules for a specific tag in external, internal as well as inline styles, then browser creates a virtual declaration list by merging all style rules tohether by following rules of cascade and then applies the merged virtual declaration list to specified tag.
